# Start-VCFUpgrade

### Synopsis
Schedule/Trigger Upgrade of a Resource

### Syntax
```
Start-VCFUpgrade
```

### Description
The Start-VCFUpgrade cmdlet triggers an upgrade of a resource in SDDC Manager

### Examples
#### Example 1
```
Start-VCFUpgrade
```
This example invokes an upgrade in SDDC Manager
