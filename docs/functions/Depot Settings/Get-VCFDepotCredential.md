# Get-VCFDepotCredential

### Synopsis
Get Depot Settings

### Syntax
```
Get-VCFDepotCredential
```

### Description
Retrieves the configuration for the depot of the connected SDDC Manager

### Examples
#### Example 1
```
Get-VCFDepotCredential
```
This example shows credentials that have been configured for the depot.
