# Start-VCFBackup

### Synopsis
Start the SDDC Manager backup

### Syntax
```
Start-VCFBackup
```

### Description
The Start-VCFBackup cmdlet invokes the SDDC Manager backup task.

### Examples
#### Example 1
```
Start-VCFBackup    
```
This example shows how to start the SDDC Manager backup
